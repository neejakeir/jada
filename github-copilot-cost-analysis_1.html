Strategy

Negotiation & savings tips

Volume discounts (20–35% off)
Organizations with 100+ seats can negotiate significant discounts. Multi-year commitments (2–3 years) unlock the deepest cuts. Always ask — list price is rarely final.
Microsoft EA bundling
GitHub is part of Microsoft's Enterprise Agreement portfolio. Bundle with Azure, M365, or Dynamics for cross-product leverage and unified billing.
Active committer billing trap
Advanced Security bills per unique committer in the last 90 days — not per seat. This number fluctuates monthly. Limit GHAS-enabled repos to control costs.
Premium request overages
$0.04/request adds up fast for heavy Chat/Agent users. Set spending limits to $0 to prevent surprise bills, or budget for a buffer (e.g., 20% over allowance).
Start with a 60-day trial
GitHub Enterprise Cloud offers a 60-day free trial with up to 50 licenses. Use it to benchmark adoption, measure Copilot usage, and build your business case before committing.
Audit unused seats
You pay for assigned seats, not active users. Review Copilot usage dashboards monthly — if a developer isn't using AI assistance, the per-seat fee is wasted spend.

Enterprise Server + Copilot compatibility

If you host your own GitHub Enterprise Server instead of using Enterprise Cloud.

Works
Copilot Business on GHES
  • Code completions in IDEs — works normally
  • Copilot Chat in IDEs — works normally
  • Requires GitHub.com account for licensing
  • Use GitHub Connect to sync licenses and avoid double billing
  • Code context sent to GitHub's cloud for AI processing
  • $19/user/month (same price as Cloud)
Not available
Copilot Enterprise on GHES
  • Custom models trained on your codebase — Cloud only
  • Knowledge bases — Cloud only
  • GitHub.com Chat integration — Cloud only
  • Copilot code review (agentic) — Cloud only
  • Requires Enterprise Cloud ($21/user/mo)
  • If code leaving your network is a concern, consider self-hosted AI alternatives (Tabby, Continue, Cline)
