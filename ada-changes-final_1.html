13. Keyboard Navigation Script

2.1.1 A Keyboard   2.1.2 A No Keyboard Trap   4.1.2 A Name, Role, Value

✗ Original
<!-- No keyboard nav script.

Bootstrap 3.3 built-in handler only
partially supported Up/Down inside
open dropdowns.

Problems:
• No Left/Right between top items
• Enter/Space didn't open dropdowns
• No Home/End in submenus
• Tab trapped focus in submenus
• No parent highlight on keyboard nav
• Bootstrap handler conflicted with
  any custom keyboard code -->
✓ Fixed — ada-menubar.js
(function ($) {
  'use strict';
  $(function () {
    $('[data-ada-menubar]').each(function () {
      var $bar  = $(this);
      var $tops = $bar.children('li');

      // Auto-apply ARIA roles
      $bar.attr('role', 'menubar');
      $tops.attr('role', 'none');
      $tops.children('a')
           .attr('role', 'menuitem');
      $bar.find('.dropdown-menu')
          .attr('role', 'menu');
      $bar.find('.dropdown-menu li')
          .attr('role', 'none');
      $bar.find('.dropdown-menu a')
          .attr('role', 'menuitem');

      // Fix href="#" → javascript:void(0)
      $bar.find('.dropdown-toggle[href="#"]')
          .attr('href','javascript:void(0)');

      // Kill Bootstrap's conflicting handler
      $bar.off('keydown.bs.dropdown.data-api');
      $bar.find('.dropdown-toggle')
        .off('keydown.bs.dropdown.data-api');

      // Highlight management
      function highlightTop($li) {
        $tops.removeClass('ada-focused');
        $li.addClass('ada-focused');
      }

      // Focus tracking on focusin/focusout
      $bar.on('focusin', function (e) {
        var $p = $(e.target).closest($tops);
        if ($p.length) {
          $tops.removeClass('ada-focused');
          $p.addClass('ada-focused');
        }
      });
      $bar.on('focusout', function () {
        setTimeout(function () {
          if (!$bar.find(':focus').length)
            $tops.removeClass('ada-focused');
        }, 10);
      });

      // Helpers: focusTopAt, openMenu, closeMenu
      // all call highlightTop() internally

      // Top-level keydown:
      //   37 Left  → focusTopAt(i - 1)
      //   39 Right → focusTopAt(i + 1)
      //   40 Down  → openMenu($li, false)
      //   38 Up    → openMenu($li, true)
      //   13/32    → openMenu($li, false)
      //   27 Esc   → closeMenu($li)

      // Submenu keydown:
      //   40 Down  → next item (wraps)
      //   38 Up    → prev item (wraps)
      //   27 Esc   → closeMenu
      //   39 Right → close, next top item
      //   37 Left  → close, prev top item
      //   36 Home  → first item
      //   35 End   → last item
      //   9  Tab   → close, resume tab

      // aria-expanded sync via
      // shown/hidden.bs.dropdown events

    });
    $(document)
      .off('keydown.bs.dropdown.data-api');
  });
})(jQuery);

Top-Level Menu Bar

KeyContextResult
Left ArrowAny top-level itemFocus + highlight moves to previous item. Wraps first → last.
Right ArrowAny top-level itemFocus + highlight moves to next item. Wraps last → first.
Down ArrowDropdown toggleOpens submenu, highlights parent, focuses first item.
Up ArrowDropdown toggleOpens submenu, highlights parent, focuses last item.
Enter / SpaceDropdown toggleOpens submenu, highlights parent, focuses first item.
EnterRegular linkNavigates to the URL.
EscapeOpen dropdownCloses submenu, returns focus + highlight to toggle.

Inside Submenu

KeyContextResult
Down ArrowSubmenu itemFocus moves to next item. Wraps to first.
Up ArrowSubmenu itemFocus moves to previous item. Wraps to last.
EscapeSubmenu itemCloses submenu, returns focus + highlight to toggle.
Right ArrowSubmenu itemCloses submenu, highlights + moves to next top-level item.
Left ArrowSubmenu itemCloses submenu, highlights + moves to previous top-level item.
HomeSubmenu itemFocus moves to first item.
EndSubmenu itemFocus moves to last item.
TabSubmenu itemCloses submenu, resumes normal tab order.
EnterSubmenu itemNavigates to the URL.
